# Control de Aportes - Restauración Poder y Vida ⛪

¡Bienvenido al sistema oficial de gestión de aportes de la congregación **Restauración Poder y Vida**! Esta aplicación móvil ha sido diseñada para facilitar el seguimiento de miembros y sus generosas contribuciones mensuales.

## ✨ Características Principales

*   **👥 Gestión de Miembros**: Registro completo de miembros con persistencia de datos local (AsyncStorage).
*   **💰 Control de Aportes**: Registro detallado de pagos con selector de miembros y fechas.
*   **✍️ Firma Digital**: Captura de firmas en pantalla mediante un panel táctil de alta precisión.
*   **📊 Estadísticas en Tiempo Real**: Visualización de recaudación total, transacciones y ranking de aportantes.
*   **📈 Historial Detallado**: Gráficos de barras (React Native Chart Kit) para ver la evolución de pagos del último año.
*   **🗓️ Semáforo Mensual**: Visualización rápida de los 12 meses (Verde para pagado, Rojo para mora).

## 🚀 Tecnologías Utilizadas

*   **React Native** + **Expo**
*   **TypeScript** (Seguridad en el tipado)
*   **Lucide React Native** (Iconografía moderna)
*   **React Navigation** (Navegación fluida entre pantallas)
*   **React Native Signature Canvas** (Firmas digitales)

## 🎨 Diseño y Aestética

La aplicación utiliza un esquema de colores **Indigo & Emerald**, diseñado con **StyleSheet** para ofrecer una interfaz limpia, moderna y con efectos de sombreado premium.

## 📦 Instalación (Desarrollador)

1. Clona este repositorio.
2. Ejecuta `npm install` para instalar dependencias.
3. Inicia el proyecto con `npx expo start`.
